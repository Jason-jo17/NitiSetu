# NitiSetu — Responsible AI Declaration

## Commitment

Acolyte AI commits to the following principles throughout the development and deployment of NitiSetu:

### 1. Human Primacy
NitiSetu generates structured assessments and recommendations. CDSCO officers retain sole authority over all regulatory decisions. The system explicitly labels every AI output as "AI-assisted analysis — subject to expert review."

### 2. Transparency
Every output includes:
- Model version and provider
- Confidence score
- Input data source
- Specific regulatory reference for each finding

### 3. Non-discrimination
- Training data audited for demographic bias
- Severity classification validated for consistency across patient age, sex, and geographic origin dimensions

### 4. Data Minimisation
- Only the minimum data required for each processing task is accessed
- Extracted text is discarded after processing unless explicitly stored by the reviewer
- PII is anonymized locally before any external API calls

### 5. No Foreign Collaboration
All development is conducted within India by the Acolyte AI team. No data, models, or code will be shared with foreign entities without IndiaAI approval.

### 6. IP Assignment
Acolyte AI agrees to assign IP of the submitted solution to IndiaAI and CDSCO as specified in the hackathon terms and conditions.

### 7. Plagiarism-free
All code and models are original or appropriately attributed. Pre-trained models (Presidio, SentenceTransformers, PaddleOCR) are open-source with compatible licenses (MIT, Apache 2.0).
