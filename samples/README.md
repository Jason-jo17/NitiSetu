# Sample Test Documents

This directory contains anonymized sample documents for testing NitiSetu features.

## Files

1. `ct04_sample.pdf` — Sample CT-04 clinical trial application (anonymized)
2. `sae_cioms_sample.pdf` — Sample SAE CIOMS-I report (synthetic)
3. `inspection_notes.jpg` — Sample handwritten inspection notes
4. `protocol_v1.pdf` — Sample protocol version 1 (for comparison)
5. `protocol_v2.pdf` — Sample protocol version 2 (for comparison)

## Usage

1. Upload via NitiSetu dashboard → Upload page
2. Select document in feature-specific pages
3. Run processing to see AI analysis results

## Note

All patient data in these documents is synthetic and does not represent real individuals.
